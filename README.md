# Portfolio

Portfolio pessoal desenvolvido com React, Vite, TypeScript e Tailwind CSS.

## Tecnologias

- React
- Vite
- TypeScript
- Tailwind CSS
- React Router DOM
- Lucide React

## Desenvolvimento

1. Clone o repositório
2. Instale as dependências com `npm install`
3. Inicie o projeto com `npm run dev`
