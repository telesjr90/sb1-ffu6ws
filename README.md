# Portfolio React + Vite

Portfolio pessoal desenvolvido com React, Vite, TypeScript e Tailwind CSS.

## 🚀 Tecnologias

- React
- Vite
- TypeScript
- Tailwind CSS
- React Router DOM
- Lucide React

## 🛠️ Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o projeto:
```bash
npm run dev
```
